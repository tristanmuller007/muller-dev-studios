# Muller Dev Studios

Site officiel de **Muller Dev Studios** — création de sites web professionnels pour artisans, commerçants et indépendants.

🌍 Basé à Bourg-en-Bresse · Disponible partout en France.

## Stack
- HTML / CSS / JavaScript pur (vanilla)
- WebGL via [cobe](https://github.com/shuding/cobe) pour le globe 3D
- Canvas API pour le flow field animé du hero
- Police : Barlow Condensed (Google Fonts)
